# 学术论文DOCX排版工具包

## 相关技能与路由
- 要**Word 排版/表格图片规整/多文档合并/期刊模板**：用本技能。
- 要**写作内容模板/短语**：用 `academic-writing-bank`。
- 要**出统计图表**：用 `academic-chart-gen`；要**架构图**：用 `academic-figure-gen`。

## 概述

本技能是 **python-docx 在学术论文场景下的增强补充**，为学术写作者提供一套经过验证的 DOCX 排版代码模板和操作策略。它不替代 python-docx 库本身，也不替代现有的 docx/word-docx 通用技能——而是在这些基础之上，专门面向学术论文等需要严密格式控制的场景进行强化。

## 🚀 5分钟快速上手（5min Quick Start）

用户说「帮我校对这一节的格式」→ 本技能先判断场景（期刊模板 / 图片管理 / 段落操作 / 文档合并 / 安全批量）→ 给出对应 python-docx 代码模板与操作策略 → 涉及批量修改时先自动备份原文件并提示你确认 → 执行安全模式（备份→分批→验证→提交）。**不自动发布、不擅自改写你的论文内容、批量改动前必须你确认。**
